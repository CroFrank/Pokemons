
# Pokemons

Hosted on github, you can visit [here](https://crofrank.github.io/Pokemons/)

### About app

Search for pokemons and see some details about them.
Pokemon info API from **https://pokeapi.co**

### I used:

- Vite for a build tool
- React framework

### Try it out

- git clone
- npm i
- npm run dev
